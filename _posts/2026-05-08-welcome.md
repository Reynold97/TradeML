This is the first post on the blog. To add another, drop a Markdown file into the
`_posts/` directory using the filename pattern `YYYY-MM-DD-title.md`, with this
front matter at the top:

```yaml
---
layout: post
title: "Your title here"
date: 2026-05-08
---
```

Everything below the front matter is plain Markdown. Push to `main` and the site
rebuilds automatically.
